# Feane - Restaurant Website Template

This is a static website template for a restaurant, named Feane.

## Features

*   Responsive design
*   Menu display
*   About Us section
*   Table booking form
*   Customer testimonials
*   Contact information

## Technologies Used

*   HTML5
*   CSS3 (with Bootstrap)
*   JavaScript (jQuery, OwlCarousel, Isotope, nice-select)

## Setup

To run this project locally, simply open the `index.html` file in your web browser.

## Hosting on GitHub Pages

This project can be easily hosted on GitHub Pages. Just push the code to a GitHub repository, and enable GitHub Pages in your repository settings.
